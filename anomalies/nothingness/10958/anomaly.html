Описание аномалии

Объект B.F.U-10958 - представляет из себя крупную книгу формата А4 с твёрдой чёрной обложкой, сделанной по стандартам всех учебных пособий в твёрдом переплёте. На обложке B.F.U-10958 (как на лицевой части обложки, так и на корешке) отсутствуют какие-либо надписи, которые могли бы указать на автора данного задачника или на его название. Почти все страницы и форзац учебника покрыты черной, не просвечивающейся плёнкой органического происхождения. По своему составу плёнка напоминает чернила головоногих моллюсков из отряда спрутовых. При попытке повредить плёнку, повреждённый участок практически сразу покрывается черной жидкостью, похожей на чернила, которая почти мгновенно засыхает в новую плёнку. Книга содержит 321 страницу, включая (предположительно) титульный лист и оглавление. На каждой странице с 5 по 236, в нижнем правом углу страницы белыми чернилами выведены трёхбуквенные инициалы. По предположениям исследователей, работающих с этим объектом, эти инициалы принадлежат людям, решившим задачи на соответствующих страницах верно. Страницы с 238 по 321 пусты и не содержат какие-либо инициалы. Последние 28 страниц с инициалами чередуются подписи «П.М.Н» и «К.А.В».
Наибольший же интерес представляет страница №237. На ней расположены условия задачи, обозначенной как «Задача Танежи». Сама страница состоит из самой обычной бумаги цвета слоновой кости с неявно выраженной фактурой. Состав бумаги не представляет особого интереса. Тест напечатан той же чёрной плёнкой, которой покрыты другие страницы. Помимо условий задачи, так же имеется поле для начала её решения.
Объект был обнаружен в 20██ году, среди задачников читального зала Архива, после показаний одного из Младших научных сотрудников, подвергнутого эффекту объекта. B.F.U-10958 был изъят из Архива и перемещён в одну из аудиторий содержания для дальнейшего исследования.
В ходе экспериментов с объектом, проводимых при помощи добровольцев из Стажёров был обнаружен и описан эффект, оказываемый объектом на человека и список правил, нарушение которых ведёт к преждевременному проявлению эффекта объекта.

Условия "Задачи Танежи"

1. Цель задачи – представить все числа от 0 до 11111 в виде операций над двумя последовательностями чисел ( {1, 2, 3, 4, 5, 6, 7, 8, 9} или { 9, 8, 7, 6, 5, 4, 3, 2, 1}).
2. Последовательности 1, 2…8, 9 и 9, 8…2, 1 не должны нарушаться при визуальной записи, все числа должны идти по порядку.
3. Разрешены операции: сложение, вычитание, умножение, деление, возведение в степень и расстановка скобок, а также конкатенация (соединение 2+ цифр в одно число). Сложные операции по типу факториала, извлечения квадратного корня и подобные им недопустимы.

Общие правила взаимодействия с объектом

1. Если читатель, изучающий B.F.U-10958 приступил к решению задачи (началом решения считается касание любой письменной принадлежностью поля для решения), то отказ от решения вызовет преждевременное проявления действия объекта.
2. Нанесение какого-либо физического вреда объекту вызовет преждевременное проявления действия объекта.
3. Попытки прибегнуть к помощи посторонних существ или объектов в ходе решения задачи (любым способом) вызовет преждевременное проявления действия объекта.
4. Все попытки решения должны быть записаны в поле под задачей. Всё написанное вне этого поля почти мгновенно поглощается бумагой.
5. В конце решения нужно поставить свои инициалы.
6. По сведениям из анонимного источника, решение задачи дарует тому, кто её решил некоторые знания, лежащие в области его интересов. Единственное условие – эти знания будут ёмким и понятным ответом на вопрос решившего.
По ходу решения задачи, плёнка постепенно отступает со страниц, освобождая пустое поле для решения. Дойти до конца страниц в ходе эксперимента не удалось. В случае нарушения правил или неверного ответа на задачу, всё основное поле очищается, а дополнительное вновь покрывается чёрной плёнкой.
Люди, попавшие под эффект B.F.U-10958 в течение нескольких недель постепенно сходят с ума, бесконечно считая вслух с 0 до 11111. Первые пару дней после появления эффекта человек ещё находится в себе и даже может дать показания.

Особые условия содержания

Объект следует содержать в ячейке хранения Архивов в специальном контейнере с защитным механизмом, допуск к нему должен иметься лишь у Старших научных сотрудников.
